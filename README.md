# Landing para GitHub Pages

Contenido listo para publicar en GitHub Pages.

## Archivos principales
- index.html
- precios.html
- privacidad.html
- terminos.html

## Publicar rapido
1. Sube esta carpeta a un repositorio dedicado (o a la carpeta docs de tu repo).
2. En GitHub: Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main y carpeta /root (o /docs si decides mover esta carpeta a docs).
